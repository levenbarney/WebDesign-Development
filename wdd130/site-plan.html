Content

Home Page

My home page will be used to showcase my rec center. It will also show all the different video games and systems I provide. It will include the location of my rec center. It will include information about pricing. It will include images of others using my facility. It will also include links to the reservation page and the frequently asked questions page.

Images for the Home page

Reservations Page

This page will allow users to see the scheduling and availability of my place. This page will allow users to make a reservation for my Gaming Rec Center. There will be a working schedule that people add too but not delete others reservations. It will include links to the Home page and the FAQ page.
